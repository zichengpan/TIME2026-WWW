# TIME 2026 Workshop Website

Official website for the **TIME 2026** - *The Second International Workshop on Transformative Insights in Multi-faceted Evaluation*, organized as part of The Web Conference 2026 (WWW 2026).

## Overview

A responsive, professional workshop website built with Jekyll featuring:

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean green color scheme with smooth transitions
- **Section Navigation**: Smooth scroll navigation between workshop sections
- **Professional Typography**: Times New Roman serif font throughout
- **Mobile Menu**: Hamburger navigation for smaller screens

## Key Sections

- **Hero Section**: Event title with background image and overlay
- **Overview**: Workshop description and key focus areas
- **Speakers**: Invited speaker profiles and information
- **Call for Papers**: Submission guidelines and key topics
- **Important Dates**: Workshop and submission deadlines
- **Organization**: Workshop organizers and committee members
- **Contact**: Contact information and additional resources

## Technology

- **Static Site Generator**: Jekyll 4.2.2
- **Styling**: SCSS with responsive design
- **No Dependencies**: Pure HTML, CSS, and JavaScript (no frameworks)
- **Browser Support**: All modern browsers (Chrome, Firefox, Safari, Edge)
